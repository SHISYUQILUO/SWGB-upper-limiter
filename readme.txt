数据处理

1. **数据清洗** - 去除NaN、无穷大等非法值
2. **去直流与去趋势** - 消除均值偏移和多项式趋势（地震漂移）
3. **重采样** - 16384Hz降采样到4096Hz
4. **带通滤波** - 保留20-1800Hz频带（Butterworth滤波）
5. **频域白化** - 除以PSD平方根，将有色噪声转为白噪声
6. **标准化** - 零均值单位方差（Z-score标准化）
7. **限幅** - 截断到[-20, 20]范围防止极端值

外加**预处理状态诊断**（检测是否已处理）和**可视化对比**（生成原始/处理后对比图）。
